Multiplicación

la multiplicación es una forma de sumar, cuando se necesita sumar varias veces un mismo número se podría utilizar la multiplicación. Para esto se selecciona un número y las veces que debe ser sumado, por ejemplo: 2+2+2=6, esto sería igual a seleccionar el 2 y multiplicarlo por 3 que es el número de veces que se repite, de ese modo la multiplicación sería 2 X 3=6.:

División

la división es la operación inversa a la multiplicación, esta se utiliza para repartir en partes iguales una cantidad que se tiene, por ejemplo: un niño tiene 6 monedas y quiere repartirlas en partes iguales con su hermano, para saber cuántas monedas le tocan a cada uno se debe hacer una división de la siguiente forma 6/2=3, esto quiere decir que le tocaría una cantidad de 3 monedas a cada niño.
